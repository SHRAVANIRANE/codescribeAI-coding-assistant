# CodeScribeAI 🤖

**CodeScribeAI** is an AI-powered coding assistant that helps developers write, debug, and understand code faster. It provides intelligent code suggestions, syntax highlighting, and code explanations across multiple programming languages.

⚠️ **Note:** This project is currently **in progress**. Features and functionality are being actively developed.

## Features

- AI-assisted code generation and suggestions
- Multi-language support (Python, JavaScript, Node.js, etc.)
- Syntax highlighting with Prism.js
- Code explanation and debugging assistance
- Chat-based interface for interactive coding help
- Dark/Light mode toggle


## Technologies Used

- Frontend: React, Tailwind CSS, Prism.js
- Backend: Python FastAPI 
- AI Integration: Ollama
- Other: Multer for file uploads, Nodemailer for emails, JWT for authentication
